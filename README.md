A **full-stack authentication system** built with **Laravel Sanctum** and **React**, featuring:

* **User registration & login** with validation (password strength, unique username/email)
* **Cookie-based authentication** (no bearer token in the frontend)
* **Email verification** with resend cooldown
* **Google OAuth button** integration
* **Frontend built with React + Redux Toolkit**
* **Shadcn UI components + TailwindCSS** for a polished, responsive interface
* **Server-side validation** with clear error messages displayed in the frontend
* **Toast notifications** for success/error feedback
* **Persistent auth state** across page reloads
